# 🚀 TEYZIX – Interactive SaaS Landing Page

A modern, responsive, and interactive SaaS Landing Page built using **React.js**, **Vite**, **Tailwind CSS**, and **Framer Motion**. This project showcases a production-style frontend with beautiful animations, responsive layouts, interactive UI components, and a clean user experience.

# 📖 About The Project

TEYZIX is a fictional SaaS platform designed to demonstrate modern frontend development skills. The application focuses on delivering an engaging user experience through smooth animations, reusable React components, responsive layouts, and visually appealing UI design.

The project follows modern frontend development practices including component-based architecture, responsive design, reusable UI components, and clean folder organization.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/teyzix-saas-platform.git
```

Move into the project

```bash
cd teyzix-saas-platform
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
